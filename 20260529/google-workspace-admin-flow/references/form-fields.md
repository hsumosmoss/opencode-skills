# 表單與 Sheet 欄位定義

## 一、公文專案啟動表（19 欄）

```
一、基本資料
├── 專案年度           [text]  必填  例：115
├── 承辦處室           [text]  必填  例：教務處、學務處、總務處
├── 專案名稱           [text]  必填  例：AI 融入教學研習
├── 承辦人             [text]  必填
├── 承辦人Email        [text]  必填
└── 協辦人員           [長文字]

二、公文資訊
├── 公文主旨           [text]
├── 來文單位           [text]
├── 公文日期           [date]
└── 公文文號           [text]

三、期限與成果要求
├── 活動日期           [date]
├── 成果繳交期限       [date]
├── 經費核銷期限       [date]
├── 是否有經費         [choice]   是 / 否 / 不確定
├── 核定或預估金額     [text]
├── 是否需要收家長或教師回覆 [choice]
├── 是否需要活動照片   [choice]
├── 是否需要成果報告   [choice]
└── 備註               [長文字]
```

## 二、專案階段日期新增表（10 欄）

```
├── 專案編號           [text]   必填  從總控表複製
├── 專案名稱           [text]   必填
├── 日期類型           [list]   必填  14 選項（見下）
├── 日期               [date]   必填
├── 提醒設定           [checkbox]  5 選項（見下）
├── 負責人             [text]   必填
├── 負責人Email        [text]   必填
├── 是否寫入待辦追蹤表 [choice] 必填  是 / 否
├── 是否建立Calendar提醒 [choice] 必填  是 / 否
├── 說明               [長文字]
└── 備註               [長文字]
```

### 日期類型 14 選項 → 任務階段對照

| 日期類型 | 自動歸入階段 |
|---------|------------|
| 校內協調會 / 第一次會議 / 活動前檢查日 / 正式活動日期 | **執行** |
| 報名開始日 / 報名截止日 / 資料回收期限 | **表單收件** |
| 採購期限 / 經費核銷期限 | **經費** |
| 照片與成果資料回收日 / 成果報告初稿期限 / 成果送出期限 | **成果** |
| 結案檢討日 | **結案** |
| 其他 | **其他** |

### 提醒設定 5 選項

| 選項 | 偏移量 |
|------|--------|
| 當天提醒 | 0 |
| 前 1 天提醒 | -1 |
| 前 3 天提醒 | -3 |
| 前 7 天提醒 | -7 |
| 前 14 天提醒 | -14 |

## 三、行政專案總控表（20 欄）

```
專案編號 | 專案名稱 | 年度 | 承辦處室 | 承辦人 | 承辦人Email
Drive資料夾連結 | 專案紀錄Docs連結 | 待辦追蹤表連結 | 成果檢核表連結
NotebookLM筆記本連結 | 專案狀態 | 備註
來文單位 | 公文文號 | 建立日期
活動日期 | 成果期限 | 經費期限 | 是否有經費
```

**專案編號格式**：`{年度}-{處室}-{MMddHHmmss}`
範例：`115-教務處-0521143025`

## 四、專案階段日期紀錄（17 欄）

```
紀錄編號 | 新增時間 | 建立者Email
專案編號 | 專案名稱
日期類型 | 日期 | 提醒設定
負責人 | 負責人Email
是否寫入待辦追蹤表 | 是否建立Calendar提醒
說明 | Calendar事件ID | 對應待辦追蹤表連結
狀態 | 備註
```

**紀錄編號格式**：`M-{yyyyMMdd-HHmmss}`
**狀態**：處理中 / 有效 / 錯誤

## 五、每案待辦追蹤表（12 欄 + 預設 10 筆）

```
任務編號 | 階段 | 任務名稱 | 任務說明
負責人 | 協助單位 | 期限 | 提醒日期
狀態 | 需要附件 | 附件位置 | 備註
```

預設 10 筆任務（T001~T010）：
1. T001 [公文] 整理公文要求
2. T002 [公文] 建立 NotebookLM 筆記本
3. T003 [籌備] 撰寫或修正計畫書
4. T004 [籌備] 建立報名或回覆表單
5. T005 [執行] 收集活動照片
6. T006 [執行] 整理簽到表或參與名冊
7. T007 [經費] 整理經費明細與憑證
8. T008 [成果] 撰寫成果報告草稿
9. T009 [成果] 檢查成果附件是否齊全
10. T010 [結案] 建立下次改進建議

**狀態下拉選單**：未開始 / 進行中 / 待確認 / 完成

## 六、每案成果檢核表（6 欄 + 預設 10 項）

```
項目 | 是否需要 | 目前狀態 | 存放位置 | 負責人 | 備註
```

預設 10 項對應 10 個子資料夾。

**目前狀態下拉**：待整理 / 進行中 / 已完成 / 不需要 / 需人工確認
