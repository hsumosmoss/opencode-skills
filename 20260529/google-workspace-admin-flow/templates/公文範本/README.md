# 公文格式範本（範本資源）

本資料夾收錄學校常用的公文 Word 範本，供「行政專案工作流」在產出公文初稿、建立專案文件時，作為**格式與用語的參照範本**。

## 範本清單

| 檔案 | 用途 |
|------|------|
| `簽.doc` | 簽呈：對上級陳述事項、請示或建議時使用 |
| `函.doc` | 公函：對外（或平行單位）行文時使用 |
| `開會通知單.doc` | 開會通知單：召開會議時通知與會人員 |

## 來源與授權

- 來源：臺北市立大學 總務處文書組「常用公文格式文字檔」公開頁面
  （<https://document.utaipei.edu.tw/p/405-1043-41467,c17.php?Lang=zh-tw>）
- 這些是公開提供的公文格式範本。實際套用時，請依**貴校公文系統與用印/簽核規定**調整單位名稱、文號、發文格式等欄位。

## 在工作流中怎麼用

1. 公文進來、用行政工作流建好專案資料夾後，把對應範本複製到該案的
   `01_計畫書與核定資料` 或 `07_公告通知與對外文字` 子資料夾。
2. 請 AI 依範本格式草擬內容（例如：「依『簽』的格式，幫我擬一份○○研習的簽呈初稿」）。
3. **務必由承辦人與主管核校後再正式簽核、用印、發文** —— AI 只負責草稿，正式公文的責任仍在人。
